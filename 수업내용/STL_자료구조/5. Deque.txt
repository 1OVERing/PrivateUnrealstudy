@ 헤더 : <queue>
@ 라이브러리 : Standard Tamplate Library ( 스탠다드 템플릿 라이브러리 )
@ 클래스 이름 : std::deque
@ 컨테이너 방식 : Sequence Container (시퀸스 컨테이너)
@ 특징 :
	ㆍDeque (Doule Ended Queue : 덱) 이라고 부른다.
	ㆍQueue에 기능적으로 추가한 자료구조이다.
	ㆍ데이터의 삽입/삭제가 빠르다.
	ㆍ추가할당시 메모리를 재할당하는 것이 아닌 띄엄띄엄 보관하고 서로를 참조하고있다.
	ㆍ랜덤 액세스 속도 : O(1)


@ 추가 설명
같은 배열형식인 vector 같은 경우 메모리를 넘어서는 할당을 할 경우 복사생성자가 호출되며
메모리를 더 크게 잡고 복사하는 재할당이 일어나는데
Deque는 메모리 추가시 기존 메모리는 그대로 가지고 추가로 기존 메모리
만큼 다른곳에 추가할당하며, 그 사이를 포인터로 연결하여 관리하게 된다.
때문에 일반 배열처럼 랜덤액세스 속도가 매우 빠르다. 
(그래도 연속된 배열인 vector가 조금 더 빠르다.)
