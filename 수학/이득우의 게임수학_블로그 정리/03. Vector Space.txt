@ Vector Space ( 벡터 공간 )
앞에서 벡터는 선형성을 가지기 때문에, 서로 더하거나 스칼라를 곱해서 늘이거나 줄일 수 있다고 하였다.
벡터가 선형성을 가지는 이유는 선형성을 담보로 만들어진 공간인 벡터 공간(Vector Space)의 원소이기 때문이다.

# Vector Space란?
벡터 공간은 선형 공간이라고도 부른다. 벡터 공간은 서로 더할 수 있고, 스칼라(Scalar)와 곱해질 수 있는 물체인 벡터들로 구성된 공간이다.
이는 첫 번째 글에서 설명한 선형성과 동일한 의미를 가진다. 벡터의 정의 자체가 선형성을 가지는 물체인 것이다.

Vector라는 것은 원칙적으로 표현할 수 없다. 다만 원점을 기준으로 하는 화살표로 시각화를 해보면
원점을 기준으로 뻗어나가는 화살표로 그릴 수 있다.
벡터공간을 시각화 한다면 원점을 기준으로 무수히 뻗어나가는 벡터가 있다고 하면
원점을 기준으로 어떠한 공간이 형성될 것이다. 이것을 벡터공간이라고 생각하면 된다.

이렇게 무수히 많이 모인 벡터에서 주목할 점은 서로 영향을 주지 못하는 벡터들이 존재한다는 것이다.
대각선으로 뻗어 나간 화살표들은 동쪽과 북쪽을 조합해서 만들 수 있을 것이다.
하지만 정북쪽을 가르키는 화살표를 표기하기 위해서 동쪽 화살표의 도움은 전혀 필요치 않다.
마찬가지로 정동쪽 화살표를 만드는데에도 북쪽 화살표는 필요 없다.
	즉, Vector2(X,Y) 라는 선이 있다고 하면 Vector2 타입으로 정북쪽, 정동쪽을 가르키는 벡터를 만든다고 하면
	정북쪽벡터(0,1) / 정동쪽벡터(1,0) 이므로 서로간의 도움 없이도 표현이 가능하다.

이렇게 서로 연관이 없는 동쪽과 북쪽 화살표를 '선형 독립' 이라고 한다. 서쪽 화살표는 동쪽에 -1스칼라곱을 하면 얻을 수 있으니 동서남북 4방향에
속하는 모든 화살표는 동쪽과 북쪽의 조합으로 모두 제작이 가능하다고 할 수 있다.
이러한 공간을 구성하는 기본 요소가 되는 벡터를 ' 기저 벡터 (Basis Vector)'라고 부른다.
여기서 특별히 크기가 1인 벡터를 특별히 ' 표준 기저 벡터 (Standard Basis Vector)'라고 부른다.

동쪽을 X 북쪽을 Y 라 하면 공간에 존재하는 벡터V는 다음과 같은 수식으로 표현할 수 있게 된다.
V = aX + bY	 스칼라( a, b )
이를 선형 조합( Linear combination: 라니어 컴버네이션 )이라고 부른다.

표준 기저벡터 X,Y에 스칼라 값을 곱해서 벡터를 생성하는 과정을 벡터를 수학적으로 스팬:Span 이라고 표현한다.
우리가 게임에서 몬스터를 스폰:Spawn 한다는 표현을 쓰듯이 말이다.
스팬된 벡터는 항상 벡터 공간 V에 속하게 된다. 다시 말하면 스팬될 수 있는 모든 벡터들이 모두 모인 곳이 벡터 공간이 되는 것이다.
이를 수학적으로 표기하면	Span { X,Y } = V	이다.

해당 벡터 공간은 두 개의 기저벡터로 구성되기에 이를 2차원 벡터 공간이라고 한다.
벡터 공간에서 차원의 수는 표쥰 기저벡터의 수를 의미한다. ( 표준 기저벡터 2개 = 2차원 , 표준 기저벡터 3개 = 3차원 )

그리고 표준 기저벡터에서 사용하는 스칼라 a,b가 사용하는 수 체계에 따라서 벡터 공간V의 크기가 정해질 것이다.
일반적으로 스칼라에서는 실수(Real Number) 체계를 사용한다. 벡터의 스칼라가 사용하는 수 체계를 체: Field 라고 하는데
어떤 체를 사용하냐에 따라서 2차원 벡터 공간의 크기와 성질이 결정된다. 모든 실수 Field를 사용하는 벡터 공간은
무한대의 크기를 가진다.

마지막으로 Vector에서 알아야 할 중요한 단어는 '직교'이다. 서로 영향을 주지 못하는 두 기저 벡터는 서로 직교한다.(Orthogonal : 오어쏘거널)고 한다.
