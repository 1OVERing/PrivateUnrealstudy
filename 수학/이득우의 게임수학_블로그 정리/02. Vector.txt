# 벡터의 정의
컴퓨터 그래픽의 기초를 이루는 원소이며, 수학과 물리학에서도 벡터는 중요한 기본 요소로 다루어진다.
수학에서의 Vector는 Vector Space ( 벡터 공간 )을 이루는 단위 원소를 의미한다.

조금 더 이해하기 쉽게 기하학적 벡터 ( Euclidean Vector)를 기반으로 설명한다.

기하학적 벡터는 '크기와 방향' 을 가지는 물체를 나타내는데 이는 일반적으로 화살표로 표현된다.
벡터의 뜻에는 운반자 라는 뜻도 있는데 벡터라는 데이터는 어떤 물체를 운반하기 위해 필요한 힘 혹은 행동에 사용한다고 할 수 있다.

화살표로 표현하는 벡터는 앞서 설명한 직선의 성질인 선형성을 가진다.

벡터의 다른 특징은 '크기와 방향' 만 존재할 뿐 위치에 대한 정보는 없다는 것이다.
게임 제작에서 주로 벡터라는 이름의 구조체를 사용해 물체의 위치를 지정하다보니 벡터가 위치를 의미하는 것으로 혼동되고 있는데
" 사실 벡터에는 위치가 없다. "

x(1,1)인 북동쪽을 가르키는 벡터가 있다면 그건 어떤 물체를 미는 힘일 뿐이고, 위치에는 전혀 관여하지 않는다.


# 위치는 존재하지 않는다.
현실 세계에서는 위치가 없는 물체는 상상할 수 없다. 현실 세계에 존재하는 어떤 물체에도 위치는 항상 존재한다.
그래서 벡터를 우리가 이해할 수 있게 시각적으로 표현한다면 원점에서 크기와 방향만 있는 화살표를 상상할 수 있다.
크기와 방향이 같다면 모든 벡터를 원점으로 옮긴다면 모두 같은 벡터가 될 것이다.

벡터는 선형성을 가지므로 가산성에 의해 서로 결합할 수 있다.

또한 벡터에 수를 곱해 벡터를 확장하거나 축소하거나 반대방향으로 향하게 만들 수 있다.
이렇게 벡터에 곱하는 수를 스칼라( Scalar ) 라고 부른다.


+
세로벡터(1,0) + 가로벡터(0,1) = 북동쪽벡터(1,1)

* ( Scalar )
세로벡터(1,0) * 2 = 길어진 세로벡터(2,0)
세로벡터(1,0) * -1 = 방향이 뒤집힌 세로벡터(-1,0)
