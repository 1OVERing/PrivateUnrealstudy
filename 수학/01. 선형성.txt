@ 선형성 (Linearity(리니어러티))의 정의
선형 대수학에서 Linear (선형)하다 라고 말할려면
SuperPosition (수퍼포지션: 중첩)	Homogeneity  (하머저니어티: 동질성) 
두가지의 조건을 만족해야 한다.

# SuperPosition (수퍼포지션: 중첩)
F ( x1 + x2 ) == F ( x1) + F ( x2)
만약 x1과 x2를 함수에 적용하기 전에 더한 것과,
함수에 각각 적용한 뒤 더한 것의 결과가 같으면,
첫번째 조건인 중첩을 만족하는 것이다.

# Homogeneity (하머저니어티: 동질성)
F( a x ) == a F( x )
만약 x에 상수 a를 곱한 값에 함수를 적용한 것이,
x를 함수에 적용한 뒤 상수 a를 곱해준 결과와 같으면,
두번째 조건인 동질성의 조건을 만족하는 것이다.

위를 하나의 수식으로 표현한다면 다음과 같다.
F ( a1 x1 + a2 x2 ) == a1 F ( x1 ) + a2 F ( x2 ) 

이러한 조건을 만족하는 연산을 선형성이 있다고 판단하게 된다.

F ( x ) == mx 같은 수식이 있다면,
선형성은 다음을 만족할 것이다.

m(a1x1 + a2x2) == a1mx1 + a2mx2 == a1 F(x1) + a2F(x2)

이와 같은 수식은 원점을 지나는 직선을 그리게 된다.
